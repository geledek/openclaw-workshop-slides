Memory, Heartbeat, Skills, and Channels — let's look at each one.

BUILDING BLOCK 1

Memory

Your agent keeps a notebook.

  • Knows who you are, what you've asked, what you prefer
  • Searches by keyword and by concept
  • Reviews and prunes — like a good PA
"An agent that cannot forget will eventually fail to understand what's important."

BUILDING BLOCK 2

Heartbeat

It wakes up. Checks in.
Without being asked.

  • Wakes every 30 minutes on its own
  • Checks inbox, reviews tasks, monitors triggers
  • Sends scheduled updates — you set it once

Analogy: voicemail box vs a person who actually monitors their phone.

BUILDING BLOCK 3

Skills

Write a job description.
It runs it forever.

  • Plain English instruction files
  • Anyone can write one — no code
  • Consistent, repeatable, shareable

Morning digest

BUILDING BLOCK 4

Before You Start: 3 Risks, 3 Rules

Malicious skills exist

341 trojanized skills found in the official marketplace. Like malicious apps — except with access to your files and API keys.

Prompt injection is real

A crafted email tricked an agent into extracting private keys in minutes. Unsolved industry-wide.

Agents can disobey

A Meta director's agent ignored "wait for confirmation" and deleted 200+ emails.

Use a managed platform

They handle security so you don't have to — especially when starting out.

Don't install random skills

Read them first, like you'd read app permissions.

Start with low-stakes tasks

Treat your agent like a new hire. Earn trust gradually.

Think of it like early smartphone apps. Powerful, useful, occasionally dangerous.

USE CASE 1 · 15 MIN

Meet Your Assistant

1 "Hi, I'm [name]. I work in [field]. I care about [2-3 topics]."
2 "What would be a useful daily briefing for someone like me?"
3 Close the chat. Reopen. Ask: "What do you know about me?"
4 Tell it something new. Close. Reopen. Ask again.

What to notice

  • It remembers across sessions — unlike any chatbot
  • Each message adds to its understanding of you
  • This is Memory working in real time

"What's the first thing you'd want your assistant to always remember about you?"

USE CASE 2 · 10 MIN

Give It a Recurring Job

1 "Every morning at 8am, send me a 5-bullet briefing on [topic]."
2 "Keep it under 200 words. Skip anything older than 48 hours."
3 "What recurring jobs do you have scheduled for me?"
4 "Run my morning briefing right now." — see the output instantly

What to notice

  • One instruction — runs forever without reminders
  • The agent doesn't need you to be present
  • This is Heartbeat in action

USE CASE 3 · 10 MIN

Trigger From a Message

1 Find an article you'd normally bookmark
2 Send just the URL to your agent — no other text
3 Watch it fetch, summarize, tag, and save
4 "Find me that article about [rough concept]"

What to notice

  • Same as sharing a link with a friend
  • No new app, no context switch
  • This is Channels + Skills + Memory working together

If your agent doesn't support this yet — that's fine. It's a skill you can add later.
